PROG 109

Introduction to Web Development

Credits: 05

Professor: Maricel Medina-Mora

Desciption: Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects.

I came into the class having very little knowledge of HTML, CSS, and JavaScript. The only little knowledge I knew was my limited knowledge of C# and how some of it's language can be used in JavaScript, but after taking this class, I learned that rarely any of that languaged carried over.

I was able to learn so much about the languages used in HTML, CSS and JavaScript since this was my first course around them. Now I can confidently create webpages thanks to this course.
